## description
- 로컬에서 테스트용으로 사용하는 nodejs-typescript 보일러 플레이트 repository

## command
- start
  - 실제 환경에서 실행
- dev
  - nodemon으로 실행, local의 api를 호출할 수 있게 NODE_TLS_REJECT_UNAUTHORIZED=0 설정
- test
  - 테스트용으로 실행

## installed library
- typeorm
- nodemon
- axios
- ts-node
- aws/sdk
